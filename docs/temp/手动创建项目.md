## 手动创建

### Server

TSRPC 框架：
```shell
npm install tsrpc
```

生成协议用的命令行工具：（一般在后端项目下使用）
```shell
npm install tsrpc-cli --save-dev
```

### Client

#### NodeJS
```shell
npm install tsrpc
```

#### 浏览器
```shell
npm install tsrpc-browser
```

#### 小程序
```shell
npm install tsrpc-miniapp
```

#### React Native
```shell
npm install tsrpc-browser
```
