# TSRPC 3.2.2 版本更新

`tsrpc` 和 `tsrpc-cli` 迎来例行更新。

## `tsrpc@3.2.2` 更新内容
### 修复
- `postDisconnectFlow` 在手动调用 `disconnect()` 时未触发的 BUG

## 升级方式

所有更新均向后兼容，您可以根据项目需要选择升级与否。

升级方式：

**后端**
```
npm i tsrpc@latest
npm i tsrpc-cli@latest -D
```

**前端**
```
npm i tsrpc-browser@latest
npm i tsrpc-miniapp@latest
```
