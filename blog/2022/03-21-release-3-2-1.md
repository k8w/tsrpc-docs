# TSRPC 3.2.1 版本更新

`tsrpc` 和 `tsrpc-cli` 迎来例行更新。

## `tsrpc@3.2.1` 更新内容
### 新增
- `preRecvDataFlow` 增加参数 `serviceName`，以便在 `preRecvDataFlow` 阶段自行解析数据格式，执行后续 API 流程。
- 支持在 `postConnectFlow` 中修改 `dataType`
### 修复
- 修复一些日志中的文字错误

## 升级方式

所有更新均向后兼容，您可以根据项目需要选择升级与否。

升级方式：

**后端**
```
npm i tsrpc@latest
npm i tsrpc-cli@latest -D
```
