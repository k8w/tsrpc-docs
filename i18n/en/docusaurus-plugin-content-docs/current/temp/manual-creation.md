## Manual creation

### Server

TSRPC framework.

```shell
npm install tsrpc
```

Command line tools for generating protocols: (generally used under back-end projects)

```shell
npm install tsrpc-cli --save-dev
```

### Client

#### NodeJS

```shell
npm install tsrpc
```

#### Browser

```shell
npm install tsrpc-browser
```

#### applet

```shell
npm install tsrpc-miniapp
```

#### React Native

```shell
npm install tsrpc-browser
```
